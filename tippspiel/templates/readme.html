Fussball Tippspiel 4 friends

Herzlich Willkommen zum Tippspiel 4 Friends. Bei dieser Anwendung handelt es sich um ein Tippspiel für Fussballturniere. Das Ziel ist es die Resultate bestmöglich vorauszusagen und dadurch möglichst viele Punkte zu erhalten. Weitere Informationen befinden sich unten. Viel Spass!

Anleitung

Auf der Seite "Tipps abgeben" können die Tipps eingetragen werden. Danach werden diese mit dem "Speichern"-Button gespeichert. Auf der Seite "Rangliste" befindet sich eine Übersicht über den Zwischenstand. Ebenfalls wird angezeigt, wie viele Punkte für den jeweiligen Nutzer zum Schluss des Turniers prophezeit sind.

Punktevergabe

Die Punktevergabe setzt sich wiefolgt zusammen:

Nutzer tippt auf richtigen Sieger oder auf Unentschieden = 5 Punkte

Nutzer tippt auf die richtige Tordifferenz = 3 Punkte

Nutzer tippt auf die richtige Anzahl Heimtore = 1 Punkt

Nutzer tippt auf die richtige Anzahl Auswärtstore = 1 Punkt

Insgesamt können somit 10 Punkte pro Spiel verdient werden. Viel Erfolg beim Tippen!
